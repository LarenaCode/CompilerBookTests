# Testeo



En este proyecto se pondrán a prueba conceptos encontrados en manuales genéricos sobre compiladores a modo de aprendizaje autodidacta y desafío personal.

Primer paso: construcción del analizador léxico.

Se diseño un analizador léxico escrito a mano, esto es, para poder comprender el funcionamiento del mismo en profundidad.
Oportunidades de mejora: Rediseñar las clases relacionadas para ofrecer portalidad.
